## 1. 文档定位

本文档结合大学英语四六级阅读的考察特征与 Claread 的输出 Schema，分析在 `cet` variant 下各组件的解析策略、密度控制、重点关注事项，并提供可直接用于 runtime prompt 注入的 `scenario_policy` 建议。

`cet` 是一个统一的 variant，不区分四级和六级。本文档提取两者的共性作为策略基础，对差异部分取**偏六级的上限**（因为能处理六级的策略向下兼容四级）。

## 2. 大学英语四六级阅读的核心特征

### 2.1 基本参数（四级/六级共性）

| 维度 | 说明 |
| --- | --- |
| CEFR 对应 | B1–B2（四级偏 B1，六级偏 B2，统一按 B1+ 处理） |
| 大纲词汇量 | 四级约 4,500 词，六级约 5,500 词（统一按 5,500 词处理） |
| 语法地位 | **非显性考点**：无独立语法题，语法通过阅读理解间接考察 |
| 句法复杂度 | 中等：从句嵌套增多，偶有长难句，但密度远低于考研 |
| 阅读理解题型 | 选词填空（Banked Cloze）+ 长篇阅读（段落信息匹配）+ 仔细阅读（选择题） |
| 文本来源 | 新闻报道、社科文章、一般学术文本、经济管理类文章 |
| 阅读分值 | 占总分 35%（248.5 分），是分值最大的单项之一 |

### 2.2 四级与六级的主要差异

| 维度 | 四级（CET-4） | 六级（CET-6） |
| --- | --- | --- |
| 词汇量要求 | 约 4,500 词 | 约 5,500 词 |
| 文章长度 | 长篇阅读约 1,000 词 | 长篇阅读约 1,200 词 |
| 句法复杂度 | 中：单层从句为主，偶有双层嵌套 | 中高：双层嵌套增多，出现少量长难句 |
| 推理深度 | 以信息提取和简单推理为主 | 开始出现中度推理，选项干扰性增强 |
| 词汇考察 | 常见多义词、基础词组搭配 | 较难词汇辨析、同义替换更隐蔽 |

两者差异是**程度差异**而非**类型差异**，核心考察逻辑一致，因此合并为统一 variant。

### 2.3 四六级阅读的核心考察逻辑

四六级与其他考试的核心区别在于两点，对四级和六级都成立：

**第一：信息定位与快速阅读是核心能力要求**

四六级是国内主流英语考试中**最强调阅读速度和信息定位能力**的考试。三种阅读题型各有侧重：

- **选词填空（Banked Cloze）**：从 15 个词中选 10 个填入文章空白处，考察**词汇辨析 + 语境判断 + 词性识别**
- **长篇阅读（段落信息匹配）**：将 10 个句子与 12–15 个段落匹配，考察**略读（skimming）+ 查读（scanning）+ 同义替换识别**
- **仔细阅读（选择题）**：2 篇文章各 5 题，考察主旨、细节、推断、词义，与传统阅读理解类似

**第二：同义替换（Paraphrasing）是核心考察机制**

四六级阅读（尤其是长篇阅读和仔细阅读）的题目选项几乎不直接使用原文原词，而是通过**同义替换**来设置正确答案。这是四六级区别于高考的重要特征，也是向雅思托福过渡的桥梁：

- 题干中的表述是原文信息的**改写**（用不同的词表达相同的意思）
- 正确选项通常是原文关键信息的**同义表达**
- 干扰选项常使用原文原词但**改变了语义关系**

## 3. 各组件在 `cet` variant 下的解析策略

### 3.1 `VocabHighlight` · 考试词汇高亮

**解析大方向**：侧重**四六级高频核心词和词性变化敏感词**。CET 用户处于从高考词汇向大学词汇过渡的阶段，核心需求是扩展词汇量和掌握常见多义词。

| 维度 | cet variant 策略 |
| --- | --- |
| **选词优先级** | ① 四六级高频词（尤其是超出高考大纲的新增词）② 选词填空常考的词性易混淆词（如 名词/形容词/副词 同根词）③ 常见多义词的非基础义项 |
| **`exam_tags`** | exam_tags 不由 LLM 输出，而是由 PostgreSQL 词典数据库自动匹配。LLM 只需尽量标注考试相关词汇，用户点击查词时，单词卡片会自动显示数据库中已标注的 exam_tag（如 `CET-4`、`CET-6`）。因此 prompt 中无需要求 LLM 输出 exam_tags，只需引导其优先选择四六级高频词即可 |
| **密度控制** | 中等偏高（6–9 个/篇）——用户正处于词汇扩展期，需要较多词汇支持。annotation_density 控制每句标注上限，以避免前端显示杂乱为主，应适当放宽；总量偏好通过 prompt 策略引导 |
| **不应标注** | 高考已掌握的基础词（如 environment, technology, advantage） |

**与高考/考研 variant 的关键区别**：

- 高考标注的是"考试大纲核心词"——侧重基础覆盖
- 考研标注的是"熟词僻义和考研难词"——侧重深度辨析
- 四六级标注的是"**扩展词汇 + 词性变化**"——侧重从基础向中高级过渡

**特别说明：选词填空对词性的考察**

选词填空是四六级特有的题型，要求考生从 15 个备选词中选择正确的词填入 10 个空。这个题型强烈依赖**词性判断能力**（判断空格处需要名词、动词、形容词还是副词）。因此 `VocabHighlight` 在标注时，应优先选择那些**词性容易混淆的同根词**（如 economy / economic / economical / economically），帮助用户建立词性敏感度。

### 3.2 `PhraseGloss` · 短语/搭配释义

**解析大方向**：侧重**高频固定搭配和同义替换中的关键表达**，兼顾选词填空中的搭配线索。

| 维度 | cet variant 策略 |
| --- | --- |
| **标注优先级** | ① 四六级高频短语动词和固定搭配（account for, result in, contribute to, in terms of）② 同义替换中的关键表达（原文用 A 表达，题目用 B 表达，两者等义）③ 报刊常见半正式表达（a growing number of, it is widely acknowledged that） |
| **`phrase_type`** | 以 `collocation` 和 `phrasal_verb` 为主，偶尔 `idiom` |
| **`zh` 释义风格** | 给出中文意思 + 同义表达提示（如 "account for = 占……比例/解释，注意与 result in 区分"） |
| **密度控制** | 中等偏高（5–8 个/篇） |

**为什么四六级场景下同义替换表达特别重要**：

四六级阅读的核心考察机制是同义替换。长篇阅读的段落匹配、仔细阅读的选项设置都大量使用改写和同义表达。标注原文中的关键短语搭配，并在释义中提供常见的同义表达，能直接帮助用户建立"同义替换敏感度"——这是四六级阅读提分的关键能力。

### 3.3 `ContextGloss` · 语境特殊义

**解析大方向**：四六级文本中**语境义场景多于高考、少于考研**，ContextGloss 的价值处于中间地带。

| 维度 | cet variant 策略 |
| --- | --- |
| **标注场景** | ① 常见词在新闻/社科语境中的特殊含义（如 address=解决, issue=问题/发行, channel=渠道）② 选词填空中需要语境判断的词义选择 ③ 仔细阅读中"词义题"的目标词 |
| **`gloss`** | 给出该词在此处的精确含义 |
| **`reason`** | 应说明为什么常见义在这里不适用，可提示"四六级常在选项中用该词的常见义设置干扰" |
| **密度控制** | 中低（2–4 个/篇） |

**与高考/考研的区别**：

- 高考极少出现语境义（1–2 个/篇），文本词汇难度低
- 考研大量出现熟词僻义（2–3 个/篇），且是核心考察方式
- 四六级处于中间——语境义开始增多，但还不是核心考察手段

### 3.4 `GrammarNote` · 语法旁注

**解析大方向**：与高考的"显性语法教学"和考研的"长难句结构分析"都不同，四六级场景下 GrammarNote 的目标是**辅助理解 + 培养阅读速度**。语法在四六级中既不是考点也不是核心门槛，而是"流畅理解的底层支撑"。

| 维度 | cet variant 策略 |
| --- | --- |
| **讲解方式** | 理解辅助为主：帮用户快速看清句子结构，提升阅读速度 |
| **术语使用** | 可使用基础语法术语（用户有高中语法基础），但不需要深入讲解规则 |
| **`note_zh` 风格** | 应包含：① 这个结构是什么 ② 它在句中的作用 ③ 快速理解的提示（如"看到 which 先找先行词"） |
| **密度控制** | 中等（3–5 个/篇）——四六级文本结构复杂度中等 |

#### 四六级重点语法结构清单

四六级的语法重点不同于高考（考规则）和考研（拆长难句），而是"**什么结构会拖慢阅读速度**"：

### 第一梯队：影响阅读速度的核心结构（必标）

| 结构类型 | 为什么影响四六级阅读 | `note_zh` 应如何写 |
| --- | --- | --- |
| **定语从句（限制性/非限制性）** | 新闻报刊文本大量使用定语从句补充信息，不识别会导致主干丢失 | 指出"which/who 引导的从句是补充说明，可先跳过抓主干" |
| **非谓语动词作状语/定语** | 新闻体大量使用分词结构压缩信息，影响理解速度 | 指出"这个分词结构的作用是什么，修饰谁" |
| **被动语态** | 学术和新闻文本大量使用被动语态，用户常对"谁做了什么"感到困惑 | 指出"动作的执行者是谁"或"被动语态在这里强调的是什么" |
| **名词性从句** | that 引导的宾语从句和主语从句是新闻报道的常见结构 | 指出"that 从句在句中做什么成分" |

### 第二梯队：常见结构（有则标）

- 状语从句（时间、原因、条件、让步）
- 并列结构（尤其是长并列导致的阅读停顿）
- 同位语（新闻体中常用同位语补充身份信息）
- 插入语（如 according to..., however, on the other hand）
- 比较结构（more...than, as...as, not so much...as）

#### `note_zh` 的写作原则

四六级 variant 下的 `note_zh` 应采用"**理解提速**"语气，与高考的"显性教学"和考研的"结构分析"都不同：

|  | gaokao | kaoyan | cet |
| --- | --- | --- | --- |
| 示例 | "这里用现在完成时 has increased。高考常考现在完成时与一般过去时的区别" | "这里的非限制性定语从句修饰整个主句。理解时可先跳过从句，把握主干后再回来读" | "which 引导的从句补充说明前面的 study，快速阅读时可先跳过，抓住主干 'The study found that...'" |
| 核心差异 | "这是什么 + 怎么考" | "结构作用 + 怎么拆解" | "**快速识别 + 提速建议**" |

### 3.5 `SentenceAnalysis` · 长难句拆解

**解析大方向**：四六级文本中长难句的密度和复杂度**介于高考和考研之间**。SentenceAnalysis 的定位是"适度拆解"，帮助用户在遇到较复杂句子时不卡住，保持阅读流畅度。

| 维度 | cet variant 策略 |
| --- | --- |
| **触发条件** | 句子包含从句嵌套 + 修饰成分、或含插入语/同位语、或总长度超过 22 词（介于高考 20 词和考研 25 词之间） |
| **拆解目标** | "快速找到主干信息"——帮用户在限时阅读中迅速抓住句子核心含义 |
| **`label`** | 应体现句子的信息层次，如"主干（研究发现）+ 定语从句（补充研究对象）+ 结果状语（说明影响）" |
| **`teach` 风格** | 先抽出主干（核心信息），再说明修饰成分各自补充了什么信息，可点出"在段落匹配题中，这句的核心信息可能被改写为……" |
| **`chunks` 粒度** | **中粒度**：主语 / 谓语 / 宾语 / 主要从句 / 主要修饰成分。不需要像考研那样拆到极细，保持清晰即可 |
| **密度控制** | 中等（2–3 句/篇）——四六级文本长难句数量有限 |

#### `teach` 示例

> "主干：A recent study has found that...（一项最新研究发现……）。that 引导宾语从句。从句中，people who regularly engage in... 的 who 引导定语从句修饰 people，说明是哪类人。主干信息是'经常做 X 的人更可能出现 Y'。在段落匹配题中，这类信息可能被改写为'X activity is linked to Y outcome'。"
>

### 3.6 `SentenceTranslation` · 逐句翻译

**解析大方向**：四六级用户的翻译需求**高于考研、低于高考**——用户能理解大部分内容，但对中等复杂度的句子需要翻译确认。同时四六级有汉译英翻译题，翻译训练具有备考价值。

| 维度 | cet variant 策略 |
| --- | --- |
| **翻译定位** | 理解辅助 + 翻译意识培养 |
| **翻译风格** | 自然意译为主，保持流畅，适度体现原文结构 |
| **翻译粒度** | 逐句翻译，较长句可适当拆分为短句以提高可读性 |
| **补充说明** | 对含同义替换的关键句，可在翻译后用括号补充"此处 A 即题目中的 B" |

## 4. 整体密度控制与优先级

### 4.1 各组件密度建议

| 组件 | 每篇数量 | 优先级 | 说明 |
| --- | --- | --- | --- |
| `VocabHighlight` | 6–9 | ★★★★ | 用户处于词汇扩展期，需要较多词汇支持 |
| `PhraseGloss` | 5–8 | ★★★★★ | **最高优先级**：同义替换和固定搭配是四六级核心考察机制 |
| `ContextGloss` | 2–4 | ★★★ | 语境义场景多于高考、少于考研 |
| `GrammarNote` | 3–5 | ★★★ | 辅助理解，提升阅读速度 |
| `SentenceAnalysis` | 2–3 | ★★★ | 适度拆解，保持阅读流畅度 |
| `SentenceTranslation` | 全量 | ★★★★ | 理解辅助 + 翻译意识培养 |

### 4.2 优先级排序

> `PhraseGloss` ＞ `VocabHighlight` = `SentenceTranslation` ＞ `GrammarNote` = `ContextGloss` = `SentenceAnalysis`
>

与其他 variant 的对比：

|  | daily_reading | gaokao | **cet** | kaoyan |
| --- | --- | --- | --- | --- |
| 最高 | 语境义词汇 + 翻译 | 语法 + 翻译 | **短语搭配（同义替换）** | 长难句拆解 |
| 高 | 短语搭配 | 考试词汇 + 搭配 | 词汇扩展 + 翻译 | 结构分析 + 语境义 + 语篇短语 |
| 中 | 语法（FonF） | 句子拆解 | 语法 + 语境义 + 句子拆解 | 翻译 + 词汇 |
| 低 | 句子拆解 | 语境义 | — | — |

## 5. Runtime Prompt 注入建议（`scenario_policy` section）

以下是建议在 `cet` variant 下注入到 runtime prompt 的 `scenario_policy` 内容。这是一个统一的 prompt，不区分四级和六级。

```
## 场景策略：大学英语四六级

用户正在备考大学英语四六级。四六级阅读的核心能力要求是信息快速定位和同义替换识别。文章来自新闻报道和社科文本，句子复杂度中等，阅读时间紧张，速度和准确性同样重要。请按以下策略调整标注：

### PhraseGloss（最高优先级）
- 这是本场景最重要的组件。四六级阅读的核心考察机制是同义替换，短语搭配是同义替换的主要载体。
- 优先标注：四六级高频搭配（account for, result in, contribute to）、报刊常见表达（a growing number of, it is widely acknowledged that）。
- zh 释义应包含同义表达提示，帮助用户建立"同义替换敏感度"。
- 每篇 5–8 个。

### VocabHighlight（高优先级）
- 用户词汇量 4,500–5,500。不要标注高考级别的基础词。
- 优先标注：四六级新增高频词、词性易混淆的同根词（economy/economic/economical）、选词填空常考词。
- exam_tags 由词典数据库自动匹配，无需 LLM 输出。只需引导 LLM 优先选择四六级高频词即可。
- 每篇 6–9 个。

### GrammarNote（中优先级）
- 目标不是教语法规则（与高考不同），也不是拆解长难句（与考研不同），而是帮用户快速识别句子结构，提升阅读速度。
- 优先标注：定语从句（影响主干识别）、非谓语动词（压缩信息）、被动语态（动作主体不清）、名词性从句。
- note_zh 应包含"快速理解提示"，如"which 从句可先跳过，抓主干"。
- 每篇 3–5 条。

### ContextGloss（中优先级）
- 四六级文本中语境义场景多于高考、少于考研。
- 优先标注选词填空和仔细阅读中需要语境判断的词义。
- reason 可提示"四六级选项常用该词的常见义设置干扰"。
- 每篇 2–4 个。

### SentenceAnalysis（中优先级）
- 四六级文本长难句密度和复杂度中等，不需要像考研那样高频触发。
- 触发条件：从句嵌套 + 修饰成分，或总长度超过 22 词。
- teach 应先抽出主干信息，再说明修饰成分，可提示"在段落匹配题中，这句核心信息可能被改写为……"。
- chunks 中粒度，保持清晰即可。
- 每篇 2–3 条。

### SentenceTranslation
- 全量翻译。
- 自然意译为主，保持流畅。
- 对含同义替换的关键句，可在翻译后补充提示。
```

## 6. 与其他 variant 的关键差异

## 7. 四六级在考试谱系中的定位

### 7.1 承上启下的桥梁角色

四六级在中国英语考试体系中扮演**承上启下**的角色：

- **承接高考**：词汇从 3,500 扩展到 4,500–5,500，语法从显性考点变为隐性支撑，阅读从"精读定位"过渡到"快速阅读 + 信息匹配"
- **衔接考研**：开始接触英美报刊文本（但难度低于考研），出现同义替换考察机制（考研更深入），句法复杂度开始提升（但未到考研的长难句密度）
- **对接雅思托福**：同义替换（paraphrasing）是四六级与雅思/托福共享的核心考察机制，段落信息匹配与雅思的 matching 题型逻辑相近

### 7.2 讲解需求分类

根据父文档的分类框架，四六级属于**第三类：能力导向 + 策略驱动**：

- 不直接考语法知识
- 词汇通过语境和同义替换间接考察
- 强调阅读策略（定位、略读、信息匹配）
- 讲解需求：语境中的词义理解辅助、同义替换识别训练、阅读速度的隐性支持
