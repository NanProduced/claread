## 1. 文档定位

> **文档分类：历史参考**。本文档是 TEM variant 的分析决策记录，结论为 TEM variant 推迟为方案 B、不合并入 kaoyan。当前 exam variant 的实现规范以 [差异化输出执行架构设计](../workflow/v3/differentiated-output-strategy-design.md) 为准。

本文档分析 TEM-4（专四）和 TEM-8（专八）的阅读考察特征，并重点评估：**能否将 kaoyan 和 tem 合并为同一个 variant（即设计文档中的 `gre_tem`）？**

## 2. TEM 考试基本参数

| 维度 | TEM-4（专四） | TEM-8（专八） |
| --- | --- | --- |
| CEFR 对应 | B2 | C1 |
| 面向人群 | 英语专业本科二年级 | 英语专业本科四年级 |
| 词汇量要求 | 约 8,000 词 | 约 13,000 词 |
| 语法地位 | **显性考点**：有独立语法与词汇选择题 | 弱化：通过阅读、翻译、改错间接考察 |
| 阅读题型 | 选择题（10 题）+ 简答题（5 题），共 20 分 | 选择题（14 题）+ 简答题（8 题），共 20 分 |
| 阅读时间 | 35 分钟 | 45 分钟 |
| 文本来源 | 一般性话题、文学作品、社会科学 | **文学作品**、学术论文、时政评论、文化议题 |
| 核心阅读策略 | 精读 + 语言素养 | **批判性阅读**  • 文学分析 |

## 3. TEM 的独特性：与考研英语的关键差异

### 3.1 相似点（可能支持合并的论据）

| 维度 | 考研 | TEM | 共性 |
| --- | --- | --- | --- |
| 句法复杂度 | 高：多层嵌套、长难句密集 | 中高–高：学术/文学文本句子复杂 | ✅ 都需要长难句拆解能力 |
| 推理深度 | 高：态度、例证、论证逻辑 | 高：态度、深层语义、作者意图 | ✅ 都强调深度推理 |
| 词汇考察方式 | 熟词僻义、词义辨析 | 熟词僻义、同义替换、高级词汇 | ✅ 都超越基础词义 |
| 用户英语基础 | 大学生，5,500+ 词汇 | 英语专业生，8,000–13,000 词汇 | ✅ 都是较高水平用户 |

### 3.2 关键差异点（反对合并的论据）

| 维度 | 考研 | TEM（尤其 TEM-8） | 差异程度 |
| --- | --- | --- | --- |
| **文本体裁** | 英美报刊议论文（The Economist, NYT） | 大量**文学作品**、文化议题、随笔 | 🟥 **本质不同** |
| **词汇量差距** | 约 5,500 词 | TEM-4 约 8,000，TEM-8 约 13,000 | 🟥 **TEM-8 是考研的 2.4 倍** |
| **词汇类型** | 报刊议论性词汇、社科词汇 | **文学性词汇**、修辞手法、文化术语 | 🟥 **词汇域完全不同** |
| **语法考察** | 非显性，语法是理解工具 | TEM-4 有**显性语法选择题**（类似高考） | 🟧 TEM-4 与考研不同 |
| **阅读目标** | 理解论证逻辑和作者态度 | 理解文学手法、作者意图、**文本美学** | 🟥 **根本不同** |
| **题型差异** | 纯选择题 | 选择题 + **简答题**（要求用英文写出答案） | 🟧 产出形式不同 |
| **用户身份** | 各专业研究生考生 | **英语专业**本科生 | 🟥 **知识背景完全不同** |

### 3.3 最关键的差异：文本体裁与阅读目标

这是决定能否合并的核心问题。

**考研英语的阅读目标是**：理解英美报刊中的论证逻辑。文本是议论性的，作者有明确的观点和论据，读者需要理解论证结构。

**TEM-8 的阅读目标是**：理解文学作品的深层含义。文本可能是小说节选、散文、书评，读者需要理解修辞手法、隐喻、象征、叙事角度等文学概念。

这导致两者在所有标注组件上的策略都有本质差异：

| 组件 | 考研的策略 | TEM-8 应有的策略（如果做） |
| --- | --- | --- |
| VocabHighlight | 熟词僻义 + 报刊难词 | 文学性词汇 + 修辞术语 + 文化特定表达 |
| PhraseGloss | 语篇连接 + 论证表达 | 文学习语 + 修辞表达 + 文化典故 |
| ContextGloss | 报刊语境中的熟词僻义 | 文学语境中的隐喻、象征、反讽等修辞含义 |
| GrammarNote | 结构分析——帮拆解长难句 | 修辞手法分析——倒装/省略/反复的文学功能 |
| SentenceAnalysis | 拆解句法层次，找主干去枝叶 | 分析句子的文学效果和修辞目的 |
| SentenceTranslation | 准确 + 句法映射 | 体现文学性和修辞美感 |

## 4. 合并可行性评估：kaoyan + tem 能否共用一个 variant？

### 4.1 结论：**不建议合并**

<aside>
⚠️

**建议不合并 kaoyan 和 tem 为同一个 variant。**

两者的文本体裁、词汇域、阅读目标和用户背景存在本质性差异，强行合并会导致两个场景都做不好。

</aside>

### 4.2 详细论证

**原因一：文本体裁本质不同，导致所有组件策略都不同**

考研文本是英美报刊议论文，TEM-8 文本大量是文学作品。这不是程度差异，而是类型差异。同一个 prompt 无法同时引导模型做好“报刊论证分析”和“文学修辞分析”。

举例：同一个句子 *"The shadows lengthened, stretching across the empty room like the fingers of some unseen hand."*

- 考研策略会拆解句法结构（分词状语 + 比喻状语）
- TEM 策略应分析“这个比喻营造了什么氛围、作者为什么选择这个意象”

两种解读角度完全不同，无法用一套 prompt 覆盖。

**原因二：词汇量差距过大，密度控制无法统一**

TEM-8 的词汇量要求（13,000）是考研（5,500）的 **2.4 倍**。这意味着：

- 考研用户看到的“难词”，TEM 用户可能已经认识
- TEM 用户需要标注的文学性词汇，考研用户完全不需要
- VocabHighlight 的选词逻辑和 exam_tags 无法统一

**原因三：GrammarNote 的角色定位冲突**

- 考研：GrammarNote 是“结构分析工具”——帮用户拆解句法层次
- TEM-4：GrammarNote 是“显性语法教学”——TEM-4 有独立语法选择题
- TEM-8：GrammarNote 应是“修辞分析工具”——倒装、省略、反复在文学中的功能

三种完全不同的 GrammarNote 角色无法合并。

**原因四：用户背景差异太大**

- 考研用户是各专业的学生，英语是工具
- TEM 用户是英语专业学生，英语是专业本身

英语专业学生对语言学术语、文学概念、修辞手法有系统学习背景，这些在考研用户中几乎不存在。如果合并，要么 TEM 用户觉得标注太浅，要么考研用户觉得标注不相关。

### 4.3 如果不合并，TEM 应该怎么处理？

有三个选项：

**方案 A：保留为独立 variant（`tem`）**

- 优点：策略可以完全贴合 TEM 的特殊需求
- 缺点：用户量可能较少（只有英语专业学生），需要单独维护一套 prompt overlay

**方案 B：延后实现，先不做**

- 优点：减少当前工作量，集中资源做好其他 variant
- 缺点：TEM 用户无法获得针对性体验

**方案 C：从设计文档中移除 TEM variant**

- 如果产品定位不服务英语专业学生，可以不做 TEM

<aside>
💡

**建议采用方案 B**：延后实现。当前优先级是 daily_reading / gaokao / kaoyan / cet / ielts_toefl，TEM 用户量小且策略复杂度高，可以放在后续迭代中单独设计。

</aside>

## 5. 对设计文档中 `gre_tem` 命名的影响

如果不合并，那么设计文档中的 `gre_tem` 命名需要调整：

### 原方案

`gre_tem` = 考研英语 + TEM（专四专八），合并为一个 variant

### 建议调整方案

| 方案 | 说明 | 优缺点 |
| --- | --- | --- |
| **方案 1（推荐）**：把 `gre_tem` 改为纯 `kaoyan` | 只保留考研英语，移除 TEM。前端显示“考研英语” | ✅ 最简洁，避免混乱。❌ TEM 用户无专属场景 |
| **方案 2**：拆为 `kaoyan`  • `tem`（延后） | 考研单独做，TEM 放在后续迭代 | ✅ 策略清晰。❌ 需要后续额外工作 |
| **方案 3**：保留 `gre_tem` 但策略偏考研 | 命名不变，但 prompt 和 policy 完全按考研设计。TEM 用户先凑合用 | ✅ 不需要改代码命名。❌ TEM 用户体验影响大 |

## 6. TEM 如果单独做 variant，策略会是什么样的？

为后续迭代预留参考，以下是 TEM variant 的策略草图：

| 组件 | TEM variant 策略要点 | 与 kaoyan 的关键差异 |
| --- | --- | --- |
| **VocabHighlight** | 文学性词汇、修辞术语、文化特定表达。词汇量门槛显著提高 | 词汇域完全不同 |
| **PhraseGloss** | 文学习语、修辞表达、文化典故。释义应包含文学功能说明 | 从论证表达转向文学表达 |
| **ContextGloss** | 隐喻、象征、反讽等修辞手法产生的特殊含义。密度应高于考研 | 从“熟词僻义”转向“修辞含义” |
| **GrammarNote** | TEM-4：显性语法教学（类似高考）。TEM-8：修辞分析工具（倒装/省略/反复的文学功能） | 角色定位完全不同 |
| **SentenceAnalysis** | 从“句法层次拆解”转向“文学效果分析”。teach 应说明“为什么作者这样写” | 拆解目标本质不同 |
| **SentenceTranslation** | 应体现文学性和修辞美感，而不只是准确 | 翻译风格不同 |
