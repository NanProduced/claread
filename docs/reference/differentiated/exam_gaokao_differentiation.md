## 1. 文档定位

本文档结合高考英语阅读的考察特征与 Claread 的输出 Schema（`VocabHighlight` / `PhraseGloss` / `ContextGloss` / `GrammarNote` / `SentenceAnalysis` / `SentenceTranslation`），分析在 `gaokao` variant 下各组件的解析大方向、密度控制和重点关注事项。

---

## 2. 高考英语阅读的核心特征

### 2.1 基本参数

| 维度 | 说明 |
| --- | --- |
| CEFR 对应 | A2–B1 |
| 大纲词汇量 | 约 3,500 词（含小学、初中、高中三个阶段） |
| 超纲词 | 极少，基本不超纲 |
| 语法地位 | **显性考点**：语法填空、短文改错均直接考语法规则 |
| 句法复杂度 | 中低：以简单句和单层从句为主，偶有双层嵌套 |
| 阅读理解题型 | 主旨大意、细节理解、推断判断、词义猜测 |
| 文本来源 | 教材改编、科普、社会生活、应用文 |
| 核心阅读策略 | 精读 + 定位，推理深度要求低 |

### 2.2 高考英语的 form-focused 特征

高考是所有主流英语考试中 **最显性的 form-focused** 考试：

- 语法填空和短文改错**直接考查语法规则**
- 阅读理解中的词义猜测题**直接考查词汇知识**
- 完形填空**同时考查词汇和语法**

这意味着高考 variant 下的解析应当：

- 语法讲解应采用**显性教学**：直接告诉用户“这是什么语法点”，而不是仅仅“帮理解”
- 词汇标注应侧重**考试高频词和固定搭配**，而不是纯粹的理解辅助
- 句子分析应帮助用户**看到考试在考什么**

## 3. 各组件在 `gaokao` variant 下的解析策略

### 3.1 `VocabHighlight` · 考试词汇高亮

**解析大方向**：优先标注高考大纲中的**核心词汇**，尤其是那些在高考中高频出现但学生容易混淆的词。

| 维度 | 高考 variant 策略 |
| --- | --- |
| **选词优先级** | ① 高考大纲重点词汇（尤其是不在初中词表中的高中新增词）② 多义词在考试中常考的特定义项 ③ 易混淆词（如 affect/effect, rise/raise） |
| **`exam_tags`** | exam_tags 不由 LLM 输出，而是由 PostgreSQL 词典数据库自动匹配。LLM 只需尽量标注考试相关词汇，用户点击查词时，单词卡片会自动显示数据库中已标注的 exam_tag（如 `GaoKao`、`CET-4`）。因此 prompt 中无需要求 LLM 输出 exam_tags，只需引导其优先选择考试高频词即可 |
| **密度控制** | 中等偏高（6–10 个/篇）——高考用户词汇量有限，需要更多词汇支持。annotation_density 控制每句标注上限，以避免前端显示杂乱为主，应适当放宽；总量偏好通过 prompt 策略引导 |
| **不应标注** | 初中已掌握的基础词（如 important, beautiful, because） |

**典型标注场景**：

- `environment`：高考高频词，拼写易错
- `adopt` vs `adapt`：易混淆词对
- `apply`：多义词，在不同语境中意义不同（申请/应用/涂抹）

### 3.2 `PhraseGloss` · 短语/搭配释义

**解析大方向**：侧重**高考高频固定搭配和短语动词**。高考完形填空和语法填空大量考查固定搭配，阅读中遇到这些搭配时应主动标注。

| 维度 | 高考 variant 策略 |
| --- | --- |
| **标注优先级** | ① 高考常考短语动词（turn out, break down, pick up）② 固定搭配（be devoted to, look forward to）③ 常考介词搭配（apply for, consist of） |
| **`phrase_type`** | 以 `phrasal_verb` 和 `collocation` 为主，偶尔 `idiom` |
| **`zh` 释义风格** | 直接给出中文意思 + 用法提示（如 “look forward to + doing sth. 期待做某事”） |
| **密度控制** | 中等（5–8 个/篇） |

**为什么高考场景下 PhraseGloss 特别重要**：

高考英语的完形填空和语法填空均大量涉及固定搭配。在阅读中遇到这些搭配时主动标注，能直接帮助用户积累考试常考的短语知识。释义中应包含用法提示（如 `to` 后接 doing 还是 do），这是高考的典型考察点。

### 3.3 `ContextGloss` · 语境特殊义

**解析大方向**：标注高考阅读中“词义猜测”题型常考察的**熟词在特定语境中的非常规含义**。

| 维度 | 高考 variant 策略 |
| --- | --- |
| **标注场景** | ① 熟词僻义（如 `address` 作“解决”而非“地址”）② 词义猜测题中的目标词 ③ 比喻/引申义 |
| **`gloss`** | 用简洁中文说明该词在此处的含义 |
| **`reason`** | 应说明为什么词典释义在这里不够用，比如“词典释义偏向‘地址’，但此处作动词表示‘解决/处理’” |
| **密度控制** | 极低（1–2 个/篇）——高考文本词汇难度较低，语境义场景不多 |

**与 daily_reading 的区别**：

在 daily_reading 中，ContextGloss 的目标是“帮理解”；在高考场景下，它的目标应转向“**帮用户意识到这类考察方式**”——即词义猜测题的视角。`reason` 字段可以隐性引导用户学会“通过上下文推断词义”的考试策略。

### 3.4 `GrammarNote` · 语法旁注

这是高考 variant 下**最重要的组件**，因为语法是高考英语的显性考点。

**解析大方向**：采用**显性语法教学**，直接告诉用户“这是什么语法现象、它为什么重要、考试怎么考”。

| 维度 | 高考 variant 策略 |
| --- | --- |
| **讲解方式** | 显性为主：直接命名语法现象 + 解释规则 |
| **术语使用** | 使用中学英语教学术语（如“定语从句”“现在分词作状语”），因为这是考试本身的语言 |
| **`note_zh` 风格** | 应包含三层：① 这是什么 ② 为什么这样用 ③ 考试中常怎么考 |
| **密度控制** | **偏高**（4–6 个/篇）——高考用户最需要语法讲解 |

#### 高考重点语法点清单

以下语法点是高考英语阅读和语法填空中的核心考察内容。LLM 不需要输出固定的标签格式，但应按照高考语法点的风格和术语来命名 `label`、撰写 `note_zh`。优先标注以下语法现象：

### 第一梯队：高频核心考点（必标）

| 语法点 | 高考考察方式 | `note_zh` 应包含的内容 |
| --- | --- | --- |
| **定语从句** | 语法填空填关系词；阅读中理解修饰关系 | 先行词是什么、为什么用这个关系词、从句修饰什么 |
| **非谓语动词** | 语法填空填正确形式（doing/done/to do）；阅读中理解句子结构 | 主语与非谓语动词的逻辑关系（主动/被动）、为什么用这种形式 |
| **时态语态** | 语法填空填动词形式；阅读中理解时间关系 | 为什么用这个时态、它表达了什么时间关系 |
| **名词性从句** | 语法填空填连接词；阅读中理解主语、宾语、表语从句 | 从句在句中做什么成分、为什么用这个连接词 |

### 第二梯队：常见考点（有则标）

| 语法点 | 高考考察方式 |
| --- | --- |
| 状语从句（时间/原因/条件/让步） | 语法填空填连词；阅读中理解句间逻辑关系 |
| 强调句 it is...that | 语法填空；阅读中识别被强调部分 |
| 虚拟语气 | 语法填空；阅读中理解假设含义 |
| 倒装句 | 语法填空；阅读中理解句子结构 |
| 并列结构 | 语法填空；阅读中理解并列关系 |

### 第三梯队：偶尔考点（见到再标）

- there be 存在句
- help + do
- 同位语
- 插入语
- 直接引语 / 间接引语

#### `note_zh` 的写作原则

高考 variant 下的 `note_zh` 应采用“**显性教学”**语气，与 daily_reading 的“理解辅助”语气明确区分：

|  | daily_reading 风格 | gaokao 风格 |
| --- | --- | --- |
| 举例 | “has scored 用了现在完成时，强调评分到目前仍有效” | “这里用现在完成时 has scored。现在完成时表示过去发生的动作对现在仍有影响。高考语法填空常考现在完成时与一般过去时的区别” |
| 差异 | 重点在“这句话怎么理解” | 重点在“这是什么语法点、规则是什么、考试怎么考” |

### 3.5 `SentenceAnalysis` · 句子拆解

**解析大方向**：显性地拆解句子结构，帮助用户**看清主谓宾**，培养“找主干、去枝叶”的阅读习惯。

| 维度 | 高考 variant 策略 |
| --- | --- |
| **触发条件** | 句子包含从句 + 修饰成分，总长度超过 20 词（高考文本的“长句”门槛较低） |
| **拆解目标** | “看清谁做了什么”——帮用户找到主语、谓语、宾语，再理解修饰成分 |
| **`label`** | 应使用中学英语教学术语，如“定语从句修饰主语 + 不定式作目的状语” |
| **`teach` 风格** | 先说主干（谁做了什么），再说修饰成分（哪部分是定语从句、哪部分是状语），最后可点出考试相关性 |
| **`chunks` 粒度** | 粗粒度：主语 / 谓语 / 宾语 / 定语从句 / 状语，不需要拆到极细的层次 |
| **密度控制** | 中等（2–4 句/篇）——高考文本中真正复杂的句子数量有限 |

#### `teach` 示例对比

|  | daily_reading 风格 | gaokao 风格 |
| --- | --- | --- |
| 示例 | “主干：They recreated（他们重现了……）。what 引导宾语从句作宾语，说的是地球生命的样子。” | “主干：They recreated（他们重现了……）。what 引导宾语从句作 recreated 的宾语。what 引导的名词性从句是高考语法填空常考点，注意 what = the thing that。” |
| 差异 | 帮用户理解内容 | 帮用户理解内容   **• 点出考试相关性** |

### 3.6 `SentenceTranslation` · 逐句翻译

**解析大方向**：高考用户对翻译的依赖较高，翻译是重要的理解通道。

| 维度 | 高考 variant 策略 |
| --- | --- |
| **翻译定位** | 主要理解通道（用户可能先看翻译再回看原文） |
| **翻译风格** | 直译为主，结构尽量贴近原文语序，方便中英对照学习 |
| **翻译粒度** | 逐句翻译，长句可拆分为多个短句 |
| **补充说明** | 可用括号补充省略的主语或逻辑关系，如“它（这部纪录片）展示了……” |

## 4. 整体密度控制与优先级

### 4.1 各组件密度建议

| 组件 | 每篇数量 | 优先级 | 说明 |
| --- | --- | --- | --- |
| `VocabHighlight` | 6–10 | ★★★★ | 高考用户词汇量有限，需要较多词汇支持 |
| `PhraseGloss` | 5–8 | ★★★★ | 固定搭配是高考核心考点 |
| `ContextGloss` | 1–2 | ★★ | 高考文本词汇难度低，语境义场景少 |
| `GrammarNote` | 4–6 | ★★★★★ | **最高优先级**：语法是高考显性考点 |
| `SentenceAnalysis` | 2–4 | ★★★ | 高考文本长句不多，但有则应拆解 |
| `SentenceTranslation` | 全量 | ★★★★★ | 高考用户对翻译依赖高 |

### 4.2 优先级排序逻辑

在高考 variant 下，各组件的优先级排序为：

> `GrammarNote` = `SentenceTranslation` ＞ `VocabHighlight` = `PhraseGloss` ＞ `SentenceAnalysis` ＞ `ContextGloss`
>

这与 daily_reading 的优先级有明显区别：

|  | daily_reading 优先级 | gaokao 优先级 |
| --- | --- | --- |
| 最高 | 语境义词汇 + 翻译 | **语法**  • 翻译 |
| 高 | 短语搭配 | 考试词汇 + 固定搭配 |
| 中 | 语法（FonF） | 句子拆解 |
| 低 | 句子拆解 | 语境义 |

## 5. 与其他 variant 的关键差异

| 维度 | `daily_reading`（intermediate） | `gaokao` |
| --- | --- | --- |
| **核心目标** | 理解内容（meaning-focused） | 掌握语言形式（form-focused） |
| **语法讲解方式** | FonF：在理解中顺带点出 | 显性教学：直接命名 + 规则 + 考点 |
| **术语使用** | 尽量避免，用直白语言 | 使用中学英语教学术语 |
| **VocabHighlight 侧重** | 语境义优先 | 考试高频词优先 |
| **PhraseGloss 释义** | 给出中文意思 | 中文意思 + 用法提示（接 doing/to do） |
| **GrammarNote 内容** | “这句怎么理解” | “这是什么 + 规则 + 考试考什么” |
| **SentenceAnalysis 粒度** | 选择性拆句，只拆复杂部分 | 积极拆句，帮看清主谓宾 |
| **翻译风格** | 自然意译 | 直译为主，贴近原文结构 |
