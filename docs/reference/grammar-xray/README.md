# Grammar X-Ray 参考资料

本目录保存 Claread Grammar X-Ray、`grammar_note` 结构化解析、中文用户英语学习语境相关的产品和教学法研究。

这些文档是研究参考，不是 API、schema、prompt 或 UI 的实现规范。需要落地时，应先转写到对应的架构、后端 contract、prompt 设计或客户端设计文档，再进入实现。

## 文档

- `chinese-exam-grammar-xray.md`：中国英语考试环境下的“语法”外延、Grammar X-Ray 解析类型与渲染类型调研。

## 使用规则

- 本目录可为 `grammar_note.xray`、Web Grammar X-Ray、render profile 和 prompt 策略提供理论依据。
- 不直接按本目录修改后端 schema 或客户端 DTO。
- 如果本文档与当前代码输出冲突，以当前代码和测试为准，再建立产品/架构任务评估是否调整。
