# Web UI Mockups

This directory stores visual direction boards for Claread Web functional UI. They are design references for implementation, not source-of-truth product specs.

## Boards

| File | Purpose |
| --- | --- |
| `web-ui-direction-board-01.png` | Multi-direction board for Reader, Reading Studio, Grammar X-Ray, and Artifact Studio |
| `desktop-layout-directions-01.png` | Desktop layout comparison board for Editorial Reader, Reading Studio, and Artifact Studio |

## Usage

- Use these boards to align layout, spacing, density, and visual hierarchy before writing UI.
- Keep the final implementation responsive. These boards are desktop-first references, not fixed pixel specs.
- Do not copy every visual element literally. Preserve the intent: paper surface, editorial reading, semantic marks, light tools.
- If a board conflicts with `apps/web/DESIGN.md`, prefer `apps/web/DESIGN.md`.

## Cleanup

These mockups can stay while Web UI direction is active. Once actual pages and screenshots become the stronger reference, either update this directory or remove stale boards.
